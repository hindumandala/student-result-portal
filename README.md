# Student Result Portal

A web application built using Django to manage student results.

## Features
- Secure login and logout
- Dashboard with total, passed, failed student count
- Add, Edit, Delete student records
- Add marks per subject
- Subject-wise grade and pass/fail status
- Overall result with failed subjects highlighted
- Search students by name
- Live deployed on Render

## Tech Stack
- Python, Django
- SQLite
- HTML, CSS

## Live Demo
https://student-result-portal-tvix.onrender.com

## Setup Instructions
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Run server: `python manage.py runserver`
